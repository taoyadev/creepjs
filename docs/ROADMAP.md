# Roadmap

This roadmap is intentionally lightweight and should evolve with real usage data.

## Phase 1 — Production Hardening

- Security hygiene (no secrets in repo, least-privilege tokens, documented secret handling)
- CI reliability (lint/typecheck/test/build gate)
- Deployment reliability (repeatable scripts + Cloudflare workflows)
- SEO + content baseline (metadata/sitemap/robots/OG image/privacy page)

## Phase 2 — Developer Experience

- Improve documentation clarity and integration recipes
- Expand playground examples and SDK snippets
- Optional: OpenAPI/Swagger docs and downloadable collections

## Phase 3 — Fingerprinting Depth

- Expand collector coverage and telemetry
- Improve UI explanations and educational guides
- Performance tuning on low-end devices (adaptive concurrency, yielding)

## Phase 4 — Advanced Features (Optional)

- Higher-level signals (consistency checks, “lies” explanations)
- Privacy/responsible-use tooling and clearer opt-outs
- Optional enterprise-oriented capabilities (carefully scoped)
