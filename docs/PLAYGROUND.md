# Playground Guide

The CreepJS Playground is an interactive environment for testing the browser fingerprinting API.

## Features

- **Live API Testing**: Test API endpoints in real-time
- **Code Examples**: View examples in JavaScript, Python, cURL
- **Interactive Forms**: Fill in parameters and see results immediately
- **Response Viewer**: View formatted JSON responses
- **Error Handling**: See detailed error messages

## How to Use

### 1. Get API Token

Visit the playground and enter your email to get a free API token.

### 2. Test Endpoints

#### Generate Fingerprint

1. Select the "Generate Fingerprint" tab
2. Your browser will automatically collect fingerprint data
3. Click "Generate Fingerprint" button
4. View the response with your unique fingerprint ID

#### Get Token

1. Select the "Get Token" tab
2. Enter your email address
3. Click "Get Token" button
4. Save your API token for future use

### 3. View Code Examples

Switch between languages to see equivalent code:

- **JavaScript (Fetch)**
- **Python (requests)**
- **cURL**

Copy the code and use it in your project.

## Best Practices

- Test with different browsers to see fingerprint variations
- Use the caching option to improve performance
- Handle errors properly in production code
- Respect user privacy (check DNT settings)

## Limitations

- Rate limited to prevent abuse
- Free tier: 1000 requests/day
- Token required for production use
