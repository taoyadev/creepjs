# HDR Support Fingerprinting

Detects High Dynamic Range display support via media queries.

## API

```javascript
const supportsHDR = matchMedia('(dynamic-range: high)').matches;
```

## Support

**HDR displays**:
- MacBook Pro 14"/16" (2021+)
- Pro Display XDR
- High-end gaming monitors
- Some flagship phones

**Distribution**: 5-8% (growing)

**Entropy**: 0.4-0.6 bits

## Privacy

**HDR support** indicates:
- Premium device ($2000+)
- Professional user (video editor, photographer)
- High income

Enables socioeconomic profiling.

**Mitigation**: Tor/Brave may report standard range only
